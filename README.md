🎮 Tic Tac Toe (Streamlit)
[🔗visit Game](https://tic--tac--toe.streamlit.app/)

A simple and interactive Tic Tac Toe game built using Python, NumPy, and Streamlit.
This project features a clean dark-themed UI and allows two players to play directly in the browser.

🚀 Features
🎯 2-player gameplay (X vs O)
🟦 Interactive clickable board
🌙 Dark theme UI
🔄 Restart game functionality
⚡ Fast and lightweight

🛠️ Tech Stack
Python
NumPy
Streamlit

🧠 Development Note
The core game logic of this project was implemented independently, while the UI design and enhancements were developed with the assistance of AI tools.

▶️ How to Run Locally
1. Clone the repository
git clone https://github.com/ashutoshkumarsen/Tic_Tac_Toe.git
cd Tic_Tac_Toe
2. Install dependencies
pip install streamlit numpy
3. Run the app
streamlit run app.py

🎮 How to Play
Player X starts first
Click on any empty cell to make a move
Players take turns
First to align 3 marks (row, column, or diagonal) wins
If all cells are filled without a winner → it's a Draw
